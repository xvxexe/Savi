# Residenze Toscana — GitHub Pages Ready

Progetto React + Vite pronto per essere caricato nella repository **Savi** e pubblicato su GitHub Pages.

## Cosa è già configurato

- base Vite impostata su `/Savi/`
- workflow GitHub Actions per build e deploy automatico
- struttura pronta per GitHub Pages
- sito demo multipagina in stile case vacanza / affitti brevi

## Se vuoi pubblicarlo nella repo `Savi`

1. Carica tutti i file della cartella nella root della repository.
2. Assicurati che la repo si chiami esattamente `Savi`.
3. Fai push su `main`.
4. Su GitHub vai in **Settings > Pages**.
5. In **Build and deployment**, seleziona **GitHub Actions**.
6. Attendi il deploy automatico.

Se in futuro cambi nome repo, aggiorna `base` in `vite.config.js`.
